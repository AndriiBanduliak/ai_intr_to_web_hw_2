Guido van Rossum

Guido van Rossum is a Dutch programmer and the creator of the Python programming language. He developed Python in 1989 while working at the Centrum Wiskunde & Informatica (CWI) in the Netherlands as a side project to fulfill the needs of his colleagues. Python was first released in 1991.

Contribution to Python

  1. Developed Python in 1989
  2. Released Python in 1991
  3. Actively contributed to Python development until 2018
  • Created the concept of "Python Enhancement Proposals" (PEPs) for language development
  • Served as Benevolent Dictator for Life (BDFL) of Python until 2018
  • Developed multiple libraries and modules for Python such as the "global interpreter lock" (GIL)
